Nesse curso é ensinado como contribuir com projeto de código aberto

Para instalar:

````console
pyhon3 -m venv .venv
source .venv/bin/activate
pip install -r requirements-dev.txt
````
Para conferiri qualidade de código:

````console
flake8
````
